Proyectos realizados a lo largo del Bachillerato

Sismografo

Descripcion del proyecto titulado sismografo realizado en el primer año de bachillerato este proyecto se basa principalmente en la creacion de un prototio de alerta sismica temprana para ayudar a la evacuacion de las personas de lugares peligrosos los desafios de este proyecto se dieron en la creacion y la busqueda de hacer un prototipo pequeño pero que a su vez sea durarero y confiable asi que decidimos diseñar una pequeña caja modelada en 3D y soldar todo el circuito a una placa para reducir el cableado y hacerlo mas accesible.

Cerradura con huella Digital

EL proyecto se basa principalmente en la creacion de una cerradura con huella digital, este proyecto consiste en un circuito que almacena las huellas digitales de los propietarios de dicha cerradura y si alguna persona ajena a esta cerradura intenta abrir con su huella inmediatamente se le niega el acceso y se almacena su huella digital en la base de datos del proyecto el mayor desafio que se presento en este proyecto fue lo dificil que es entrenar el sensor de huella digital para que este reconociera las huellas digitales, este desafio fue superado gracias al arduo trabajo que implementamos en C++.
